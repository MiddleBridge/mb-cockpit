# Git Commands - Commit and Push

## Quick Commands (Copy & Paste)

### Windows (CMD/PowerShell):
```bash
git add -A && git commit -m "Add trip expenses, evidence, and reimbursement features" && git push
```

### Linux/Mac/Git Bash:
```bash
git add -A && git commit -m "Add trip expenses, evidence, and reimbursement features" && git push
```

## Detailed Commands (Step by Step)

```bash
# 1. Add all changes
git add -A

# 2. Commit with message
git commit -m "Add trip expenses, evidence, and reimbursement features"

# 3. Push to remote
git push
```

## Using the Script Files

### Windows:
Just double-click `commit-and-push.bat` or run:
```cmd
commit-and-push.bat
```

### Linux/Mac:
```bash
chmod +x git-commit-push.sh
./git-commit-push.sh
```
